Commentaire :


La Figure 1 est une capture de l’interface de PyCharm, un environnement de développement spécialement dédié au langage Python. Ce projet repose sur le framework Flask qui permet de créer des applications web en python.

A gauche de la figure nous pouvons voir l’arborescence, le dossier template pour le frontend (HTML), le dossier static pour tous les styles et le dossier controller pour les fichiers Python et un script SQL, que l’on doit utiliser à chaque fois que l’on fait un projet avec Flask pour pouvoir se retrouver plus facilement.

Dans la partie centrale de l’image, plusieurs éléments sont soulignés en rouge. Ils mettent en évidence la maîtrise du langage Python avec l’utilisation du framework Flask, ainsi que l’écriture de requêtes SQL pour interagir avec la base de données. Par exemple, on voit une requête d’insertion (INSERT INTO ...) utilisant la fonction execute() d’un curseur SQL, ainsi que la récupération de données envoyées depuis un formulaire HTML avec “request.form.get()”.

Ces éléments illustrent aussi la capacité à écrire un back-end avec Flask, à gérer l’intégrité des données (par des vérifications sur la longueur des commentaires ou l’usage de current_timestamp pour la date actuelle), et à établir un lien fonctionnel avec des templates HTML via la fonction render_template(...), qui permet de transmettre les informations dynamiques à afficher à l’utilisateur.
